# 📱 Smartphone Review Regression Analysis Project

Regression analysis of a **50,000-record smartphone review dataset** to examine how camera, battery, performance, design, and display ratings influence overall customer ratings.

## 📊 Project Overview

This project analyzes a **mobile phone review dataset containing 50,000 observations and 25 variables** to understand how different smartphone specifications influence overall customer ratings.

## Dataset Information

The dataset includes information on:

- Customer demographics  
- Smartphone specifications  
- Pricing information  
- Textual review content  
- Metadata related to reviews  

## Key Variables

The analysis focuses on the following smartphone specification ratings:

- Camera Rating  
- Battery Life Rating  
- Performance Rating  
- Design Rating  
- Display Rating  

Other important variables include:

- Overall Rating  
- Price  
- Model  
- Review Length  
- Word Count  
- Helpful Votes  

## Objective

The main objective of this project is to analyze how smartphone specifications impact customer ratings and determine which features have the strongest influence on overall satisfaction.

## Tools Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-Learn  

## Conclusion

The analysis shows that **smartphone performance, battery, and camera ratings have the strongest influence on overall customer ratings**, while textual review characteristics such as word count or review length have a weaker relationship with ratings.
